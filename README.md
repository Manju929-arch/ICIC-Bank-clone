# ICICI Bank Clone (UI/UX Only) 💳

## 📌 Overview

This project is a **UI/UX clone of ICICI Bank**, focused purely on **design, layout, and user experience** without any JavaScript or backend logic.

The goal is to replicate a **clean, modern banking interface** using only HTML and CSS, showcasing strong frontend design skills.

---



## 🛠️ Tech Stack

* **HTML5** → Structure
* **CSS3** → Styling & Layout

  * Flexbox
  * Grid
  * Animations & Transitions

---

## 🎯 Design Principles Used

* **Consistency** → Same color palette and typography
* **Hierarchy** → Important elements stand out clearly
* **Accessibility** → Readable fonts and contrast
* **Minimalism** → Clean and uncluttered UI
